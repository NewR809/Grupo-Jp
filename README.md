# API Financiera para Power BI

Esta API permite consultar e insertar datos de gastos e ingresos desde una base de datos MySQL en Railway. Compatible con Power BI.

## Endpoints

- `/consultar_gastos` (GET)
- `/consultar_ingresos` (GET)
- `/insertar_gasto` (POST)
